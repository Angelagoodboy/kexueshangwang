## 5.⚠️注意事项



Clash for Windows 程序在不正常退出时，如电脑重启、蓝屏等，会造成电脑无法正常访问互联网，此时需要重新打开Clash fow Windows 才可以恢复正常，懒人若有科学上网需求请使用 VPN(傻瓜式) 科学上网
